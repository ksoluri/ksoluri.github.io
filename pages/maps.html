Quick embed reference — delete this section before publishing

Google My Maps

  1. Open your map → Share → Embed on my site
  2. Copy the src="..." URL from the iframe snippet
  3. Paste it into the featured iframe above
  4. Take a screenshot for the gallery card thumbnail

ArcGIS Online

  1. Open your item → Share → Embed in website
  2. Copy the embed URL from the dialog
  3. Paste into the featured iframe src
  4. Or link the card directly to your ArcGIS item page

RStudio (leaflet / tmap)

  1. htmlwidgets::saveWidget(map, "pages/my-map.html")
  2. Add my-map.html to your GitHub repo
  3. Set the card link to href="my-map.html"
  4. Use a screenshot PNG as the thumbnail image
